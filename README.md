# Alurkerja

## Requirement

- pnpm ^8.7.4
- node 18

## Run Locally

Clone the project

```bash
  git clone https://github.com/shaddamalbz/alurkerja.git
  cd alurkerja
  pnpm i --frozen-lockfile
```

Running development app

```bash
  pnpm build:ui
  pnpm dev:docs
```

Running development packages using storybook

```bash
  pnpm dev:ui
```
